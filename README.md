# 漢字フラッシュカード

## 概要
漢字と意味をフラッシュカード形式で学習できる  シンプルな Web アプリです。
HTML / CSS / JavaScript を使用しています。

---

## 機能
- 漢字フラッシュカードの表示
- カードクリックで裏返し表示
- 次のカードへ移動
- 新しい漢字カードの追加
- localStorage によるデータ保存
- 保存データの全削除

---

## 使用技術
- HTML
- CSS
- JavaScript

---

## ファイル構成
index.html
style.css
script.js

---

## 使い方
1. `index.html` をブラウザで開く  
2. カードをクリックして意味を確認  
3. 「次へ」ボタンで次のカードを表示  
4. 漢字と意味を入力して「追加」
